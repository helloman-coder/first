# 代付分享系统

这是一个简单的代付分享系统，允许用户创建代付订单并分享链接给他人进行支付。

## 功能特点

- 创建代付订单
- 生成可分享的代付链接
- 倒计时支付功能
- 订单状态管理

## 技术栈

- Node.js
- Express.js
- HTML/CSS/JavaScript

## 项目结构

```
代付项目/
├── server.js         # 后端服务
├── package.json      # 项目配置
├── README.md         # 项目说明
└── public/           # 静态文件目录
    ├── index.html    # 首页
    └── pay.html      # 代付页面
```

## 安装与运行

1. 确保已安装 Node.js (推荐 v14+)
2. 克隆或下载项目代码
3. 进入项目目录并安装依赖：
   ```
   npm install
   ```
4. 启动服务器：
   ```
   npm start
   ```
5. 在浏览器中访问 `http://localhost:3000`

## 使用说明

1. 在首页输入代付金额和订单描述
2. 点击"创建代付订单"按钮
3. 复制生成的链接并发送给朋友
4. 朋友点击链接后可以看到代付信息和倒计时
5. 点击"知道了"按钮跳转到支付页面

## 接口说明

### 创建订单
- GET `/create-order?amount=金额&desc=描述`
- 返回订单ID和支付链接

### 代付页面
- GET `/pay/:orderId`
- 显示订单详情和倒计时

### 支付成功回调
- GET `/success/:orderId`
- 更新订单状态为已支付

### 取消/超时
- GET `/cancel/:orderId`
- 更新订单状态为已取消

## 注意事项

1. 当前版本为演示版，订单数据存储在内存中，重启服务后会丢失
2. 实际部署时需要接入真实的支付网关
3. 建议添加HTTPS支持以提高安全性
4. 生产环境中应使用数据库存储订单数据
